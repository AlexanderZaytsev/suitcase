Getting started

Please note, suitcase only runs on ruby >= 1.9.

$ gem install suitcase

In your Gemfile:

gem 'suitcase'

Basics

Configuration

Suitcase.configure do |config|
  config.hotel_api_key = "your_api_key_here"
  config.hotel_cid = "55505"
end

Usage

hotel = Suitcase::Hotel.find(id: 123904)

Here, hotel is a Suitcase::Hotel object the has the schema defined here.

Also, consider checking out the User flow page on the wiki for information on using the gem in a real-world setting.

Contributing

If you make any changes to suitcase, please consider creating a pull request to the main repository if the request meets the following requirements:

  1. Well-tested (I use RSpec, may be moving over to minitest soon)
  2. Well-styled as much as possible (GitHub's Ruby Styleguide)
  3. Compliant (it should feel like the already existing code).
